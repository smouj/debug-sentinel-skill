Skill Debug Sentinel para OpenClaw.

## Qué Soluciona
Monitorea aplicaciones en busca de anomalías en tiempo de ejecución y vulnerabilidades de seguridad.

## Características
- Activación automática cuando se detectan tareas relevantes
- Resultados profesionales y listos para producción
- Enfoque primero en seguridad
- Soporte para reversión

## Uso
/debug-sentinel

## Ejemplo
```
Analyze code for security issues.
```

## Instalación
Esta habilidad está disponible automáticamente en OpenClaw.
